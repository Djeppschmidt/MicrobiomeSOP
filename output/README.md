# Output Directory

This directory contains all analysis outputs generated by the pipeline:

- `figures/` - Visualization outputs (PNG/PDF)
- `tables/` - CSV/TSV data tables
- `*.rds` - R data objects for intermediate results and final phyloseq objects

The following key files are generated:
- `phyloseq_object.rds` - Initial phyloseq object
- `phyloseq_filtered.rds` - Quality-filtered phyloseq object
- `taxonomy.rds` - Taxonomy assignment table

Output files are excluded from version control via .gitignore.
